### jlab-flutter-development-kit

A powerful CLI tool for scaffolding and managing Flutter monorepo projects.

**Features:**
- Project initialization and management
- Feature and page generation
- API integration with code generation
- Setup for flavors, Firebase, PowerSync, and more
