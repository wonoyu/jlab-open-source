# API Integration Tutorials

Step-by-step tutorials for learning how to use the wdev API integration system.

:::info
Tutorials are designed for beginners. Follow along from start to finish to learn the fundamentals of API integration with wdev_cli.
:::

## Tutorial List

| Tutorial | Duration | Level | Description |
|----------|----------|-------|-------------|
| [Getting Started](./getting-started) | 10 min | Beginner | Create your first API service |
| [Creating Your First API Service](./getting-started) | 15 min | Beginner | Complete guide to adding a new API service |
| [Adding Endpoints](./defining-endpoints) | 15 min | Beginner | Add and configure API endpoints |
| [Generating API Clients](./generating-clients) | 10 min | Beginner | Generate Retrofit API clients |
| [Injecting Services](./injecting-services) | 15 min | Intermediate | Inject services into feature pages |
| [Advanced Integration](./advanced-integration) | 20 min | Advanced | Offline-first with PowerSync |

## Prerequisites

Before starting these tutorials, ensure you have:

- Flutter SDK installed (v3.0+)
- wdev_cli installed globally
- A Flutter monorepo project
- Basic understanding of REST APIs
